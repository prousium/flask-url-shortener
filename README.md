A simple web-based URL shortener built with Flask and SQLite.

## Features

- Convert long URLs into short ones
- Automatic short-code generation
- Redirect to original URL
- JSON API support

## Tech Stack

- Python
- Flask
- SQLite (via SQLAlchemy)
- HTML (Jinja templates)
